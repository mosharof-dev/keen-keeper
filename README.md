# 🌿 Keen Keeper

> A beautifully designed personal relationship management dashboard to track, tend, and nurture the connections that matter most in your life.

[![Live Demo](https://img.shields.io/badge/Live%20Demo-Vercel-black?style=for-the-badge&logo=vercel)](https://keen-keepe.vercel.app/)

## 📖 About The Project

Keen Keeper is a modern web application built to help users manage their personal networks. It provides a visual dashboard to track interactions (calls, texts, meetups), monitor relationship health (on-track, overdue), and view interaction histories over time. This project was developed as a comprehensive assignment to demonstrate proficiency in modern full-stack web development using Next.js.

## ✨ Key Features

- **📊 Dynamic Dashboard:** Real-time calculation of total friends, interaction counts, and relationship status.
- **🍩 Interactive Analytics:** Visual representation of interaction types using beautiful Recharts donut charts.
- **⏳ Timeline History:** A dedicated timeline view powered by React Context API to track global interaction states.
- **⚡ Next.js App Router:** Optimized architecture using a mix of Server Components for fast page loads and Client Components for interactivity.
- **📱 Responsive UI:** Pixel-perfect, mobile-first design implemented with Tailwind CSS for a seamless experience across all devices.

## 🛠️ Tech Stack

- **Framework:** [Next.js (App Router)](https://nextjs.org/)
- **Library:** [React.js](https://react.dev/)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **State Management:** React Context API
- **Data Visualization:** [Recharts](https://recharts.org/)
- **Icons:** React Icons
- **Deployment:** Vercel
